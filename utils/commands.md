## Frequent Command
### To install mkdocs:

```shell
pip install mkdocs
pip install mkdocs-material
```

### To build site:

```shell
mkdocs build
```

### to preview theme:

```shell
mkdocs serve --watch-theme
```

### To deploy site:

```shell
mkdocs gh-deploy --force
```

### to kill preview program:
```shell
ps -fA | grep python
```
